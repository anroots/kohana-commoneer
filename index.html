kohana-commoneer v1.4 by anroots

A Kohana module providing commonly used helpers and organizational classes for faster development.

This is a Kohana module that aims to give developers (mainly myself) more predefined, reusable methods and extensible classes.

Dependencies

Kaltar Notify (https://github.com/kaltar/Notify), standard bundled Kohana modules

Install

Standard Kohana module install

Documentation

Activate the module and Kohanas userguide to see the documentation. You'll have to rely on the API documentation (phpdoc comments inside files), since a more decent version with examples is yet to be written.
